## `let` + `const`

```jsx
let age = 20;
const name = "Alice";
let age = 20;
age = 21;

console.log(age); // 21

```

let 表示：这个变量之后可以重新赋值。

const 表示：这个变量不能重新赋值。

```jsx
// 默认用 const
const userName = "Alice";

// 只有值之后会变化，才用 let
let count = 0;
count = count + 1;
```

const 不能重新指向另一个值，但如果它保存的是对象或数组，里面的内容可以变。

```jsx
// 定义一个对象 user
const user = {
  name: "Alice",
  age: 20
};

user.age = 21;

console.log(user); // { name: "Alice", age: 21 }

```

但是不能进行如下操作：

```jsx
const user = { name: "Alice" };

user = { name: "Bob" }; // 报错

```

数组同理：

✔️

```jsx
const numbers = [1, 2, 3];

numbers.push(4);

console.log(numbers); // [1, 2, 3, 4]

```

❌

```jsx
const numbers = [1, 2, 3];

numbers = [4, 5, 6]; // 报错

```

<aside>
💡

**const 限制的是变量名不能重新绑定，不是里面的内容永远不能变。**

</aside>

# 循环

- [x]  *让代码重复执行，批量处理数字、字符串、数组。*

## `for` 循环

```jsx
for (let i = 0; i < 5; i++) {
  console.log(i);
}
```

```jsx
// 计算1到100的和
let total = 0;

for (let i = 1; i <= 100; i++) {
  total = total + i;
}

console.log(total); // 5050
```

## 遍历数组

```jsx
const fruits = ["apple", "banana", "orange"];

for (let i = 0; i < fruits.length; i++) { //数组下标从 0 开始。
  console.log(fruits[i]);
}
// fruits[0] // "apple"
// fruits[1] // "banana"
// fruits[2] // "orange"
```

## **`for...of`  循环**

如果只想拿到数组里的每一项，for...of 更好读：

```jsx
const fruits = ["apple", "banana", "orange"];

for (const fruit of fruits) {
  console.log(fruit);
}
```

```jsx
const word = "JS";

for (const letter of word) {
  console.log(letter);
}
//J
//S
```

for...of 很适合：

- 遍历数组
- 遍历字符串
- 不关心下标，只关心每个值

## `while` 循环

只要条件成立，就一直执行。

```jsx
let count = 1;

while (count <= 5) {
  console.log(count);
  count++;
}
```

和 `for` 很像，只是写法不同。

通常：

- 明确知道循环次数：用 `for`
- 不确定循环多少次，只知道结束条件：用 `while`

```jsx
let money = 10;

while (money > 0) {
  console.log(`还剩 ${money} 元`);
  money = money - 3;
}
```

## **`break` 和 `continue`**

break：立刻结束整个循环。

```jsx
for (let i = 1; i <= 10; i++) {
  if (i === 5) {
    break;
  }

  console.log(i);
}
```

continue：跳过本次循环，进入下一次。

```jsx
for (let i = 1; i <= 5; i++) {
  if (i === 3) {
    continue;
  }

  console.log(i);
}
```

```jsx
// 打印数组中大于等于 60 的分数：

const scores = [45, 80, 66, 59, 90];

for (const score of scores) {
  if (score >= 60) {
    console.log(score);
  }
}

//统计及格人数
const scores = [45, 80, 66, 59, 90];

let passedCount = 0;

for (const score of scores) {
  if (score >= 60) {
    passedCount++;
  }
}

console.log(passedCount); // 3
```

## 总结

```jsx
// 计数循环
for (let i = 1; i <= 10; i++) {}

// 遍历数组
for (const item of array) {}

// 累加
let total = 0;
for (...) {
  total = total + value;
}
```

## **返回值 return**

console.log() 是打印结果。

return 是把结果交给函数外面继续使用。

```jsx
function add(a, b) {
  return a + b;
}

const result = add(3, 5);

console.log(result); // 8
```

<aside>
💡

console.log() // 给人看
return        // 给程序继续用

</aside>

**return 会结束函数**

```jsx
function checkAge(age) {
  if (age < 18) {
    return "未成年人";
  }

  return "成年人";
}

console.log(checkAge(16)); // 未成年人
console.log(checkAge(20)); // 成年人
```

一旦执行了 return，函数后面的代码就不会继续执行。

## 默认参数

如果调用函数时没有传值，可以设置默认值：

```jsx

}

console.log(greet()); // 你好，游客
```




```jsx
const users = [
```

## `reduce` 把数组汇总成一个结果

### 计算总和：

```jsx
const nuumbers = [10, 20, 30];
const total = number.reduce((sum, number) => {
	return sum + number;
},0);
console.log(total); // 60

//简写
const total = numbers.reduce((sum, number) => sum + number, 0);
```

<aside>
💡

sum : 累计值

number：当前元素

0 ： 初始值

</aside>

```jsx
// 计算购物车总价
const cart = [
  { name: "Book", price: 30, count: 2 },
  { name: "Pen", price: 5, count: 3 },
  { name: "Bag", price: 100, count: 1 }
];

const total = cart.reduce((sum, item) => {
  return sum + item.price * item.count;
}, 0);

console.log(total); // 175
```

# JSON

JSON 全称是 JavaScript Object Notation。

可以先把它理解成：

**一种用字符串表示数据的格式**

```jsx
// JS 对象
const user = {
  name: "Tom",
  age: 18,
  isStudent: true
};
JSON 字符串
'{"name":"Tom","age":18,"isStudent":true}'
```

<aside>
💡

主要区别：

JS 对象：代码里的数据结构
JSON：传输或保存用的字符串格式

JSON 里属性名必须用双引：

```jsx
{
  "name": "Tom",
  "age": 18
}
```

</aside>

## **JSON.stringify()**

把 JS 对象变成 JSON 字符串。

```jsx
const user = {
  name: "Tom",
  age: 18,
  isStudent: true
};

const jsonText = JSON.stringify(user);

console.log(jsonText);
console.log(typeof jsonText); // string

//输出
{"name":"Tom","age":18,"isStudent":true}

/////////////////////
const users = [
  { id: 1, name: "Tom" },
  { id: 2, name: "Alice" }
];

const jsonText = JSON.stringify(users);

console.log(jsonText);

```

## **JSON.parse()**

把 JSON 字符串转成 JS 对象。

```jsx
const jsonText = '{"name":"Tom","age":18}';

const user = JSON.parse(jsonText);

console.log(user);
console.log(user.name); // Tom

const jsonText = '[{"id":1,"name":"Tom"},{"id":2,"name":"Alice"}]';

const users = JSON.parse(jsonText);

console.log(users[0].name); // Tom
```

<aside>
💡

前端和后端通信时，数据不能直接传 JS 对象。

它通常会先变成 JSON 字符串，再通过 HTTP 发送

JSON不支持：
undefined
function
symbol

</aside>

```jsx
// 保存购物车
const cart = [
  { name: "Book", price: 30, count: 2 },
  { name: "Pen", price: 5, count: 3 }
];
const cartText = JSON.stringify(cart);

console.log(cartText);

const parsedCart = JSON.parse(cartText);

const total = parsedCart.reduce((sum, item) => {
  return sum + item.price * item.count;
}, 0);

console.log(total);
```

## 重点

前端代码里用对象，网络传输和本地存储常用 JSON 字符串。

JS 对象 / 数组 → JSON 字符串

```jsx
JSON.stringify(jsObject)
```

JSON 字符串 → JS 对象 / 数组

```jsx
JSON.parse(jsonString)
```
