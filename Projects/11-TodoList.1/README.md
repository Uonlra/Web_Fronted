# DOM Todo App 学习项目

## 学习目标

用 vibe coding 的方式，从 HTML、CSS、JavaScript 基础开始，一步一步做出 DOM 版本 Todo App。

## 当前步骤：第 1 步，HTML 和 CSS 从头开始

本步骤只做最基础的页面结构，不实现 Todo 交互。

### 当前需要实现的目标

1. 理解一个 HTML 文件的基本结构。
2. 学会使用常见 HTML 标签：`main`、`header`、`section`、`h1`、`p`、`form`、`label`、`input`、`button`、`ul`、`li`。
3. 学会用 `link` 标签连接 CSS 文件。
4. 学会 CSS 最基础的写法：选择器、颜色、间距、字体、边框。
5. 学会用 `script` 标签连接 JavaScript 文件。
