# 交互式日期选择器 UI（HTML + CSS + JavaScript）

这是一个前端练习项目，目标是实现可交互的日期选择器界面。
当前版本已支持切换年/月、选择具体日期，并把结果同步到输入框。

## 功能说明

- 支持上一月、下一月切换
- 支持通过下拉框切换月份和年份
- 支持点击某一天进行日期选择
- 选中日期会回填到输入框（格式：dd/mm/yyyy）
- 支持 Today 快捷定位到今天
- 页脚显示当前选中日期文本
- 支持展开/收起日历面板
- 支持移动端基础适配

## 项目文件

- index.html：日期选择器结构与交互控件
- style.css：主题样式、状态样式、动画与响应式规则
- script.js：日期渲染、年/月切换、日期选择与输入框同步逻辑

## 使用方式

1. 浏览器直接打开 index.html。
2. 通过顶部左右按钮或下拉框切换年/月。
3. 点击日期单元格完成选取。
4. 查看输入框和底部 Selected 文本的同步结果。

## 实现要点

- 日期网格按当前视图月份动态生成。
- 每次切换年/月会重新渲染日历。
- today 与 active（选中）状态使用不同样式区分。
- 年份选项围绕当前年份自动生成，便于快速跳转。

## 后续可扩展

1. 支持键盘方向键选择日期。
2. 支持点击组件外区域自动关闭面板。
3. 支持输入框手动输入并反向校验。
4. 支持日期范围选择（开始/结束日期）。
