# 交互式日期选择器 UI（HTML + CSS + JavaScript）

这是一个原生前端练习项目，目标是实现一个可交互、可切换年月、可选择日期的日期选择器界面。
当前版本已支持切换年/月、选择具体日期、自动回填输入框，并补充了更完整的关闭与键盘交互。

## 项目预览

这是一个卡片式日期选择器组件，包含：

- 顶部标题与输入框区域
- 可展开的日历面板
- 年/月切换控件
- 日期网格
- `Today` 快捷按钮与选中结果展示

这个项目适合作为练习以下前端基础能力的小作品：

- HTML 结构组织
- CSS 视觉设计与响应式适配
- JavaScript DOM 操作
- 组件状态管理与重新渲染思路
- 基础可访问性处理

## 技术栈

- HTML5
- CSS3
- JavaScript（Vanilla JS）

## 功能说明

- 支持上一月、下一月切换
- 支持通过下拉框切换月份和年份
- 支持点击某一天进行日期选择
- 选中日期会回填到输入框（格式：`dd/mm/yyyy`）
- 支持 `Today` 快捷定位到今天
- 页脚显示当前选中日期文本
- 支持展开/收起日历面板
- 支持点击组件外区域自动关闭面板
- 支持按 `Esc` 关闭日历面板并返回焦点
- 选择日期或点击 `Today` 后自动收起面板
- 支持移动端基础适配

## 项目文件

- `index.html`：日期选择器结构与交互控件
- `style.css`：主题样式、状态样式、动画、按钮反馈与响应式规则
- `script.js`：日期渲染、年/月切换、日期选择、面板开关与键盘/点击交互逻辑

## 使用方式

1. 浏览器直接打开 `index.html`。
2. 点击右侧日历按钮展开或收起面板。
3. 通过顶部左右按钮或下拉框切换年/月。
4. 点击日期单元格完成选取。
5. 查看输入框和底部 `Selected` 文本的同步结果。
6. 也可以点击 `Today` 快速返回今天。

## 实现要点

- 日期网格按当前视图月份动态生成，固定使用 42 个格子渲染日历布局。
- 每次切换年/月或重新选择日期时会重新渲染日历。
- `today` 与 `active`（选中）状态使用不同样式区分。
- 使用统一的点击委托处理日期按钮，而不是给每个按钮单独绑定事件。
- 年份选项围绕当前年份自动生成，范围由顶部 `config` 配置控制。
- 面板开关状态统一由 `setCalendarOpen()` 管理，便于维护 `aria-expanded` 和可访问性标签。
