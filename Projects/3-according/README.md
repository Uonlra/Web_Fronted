# Personal Portfolio

一个使用 HTML、CSS 和 JavaScript 编写的响应式个人作品集练习项目。

## 项目简介

这个项目实现了一个多页面前端作品集网站，包含首页、项目页、文章页和联系页。布局参考了移动端和桌面端的线框图，重点放在语义化结构、响应式布局和清晰的信息层级上。

## 主要功能

- 响应式页面布局，适配移动端和桌面端
- 多页面导航结构
- 首页项目列表可跳转到项目页对应条目
- 深色模式和浅色模式切换
- 基础 SEO Meta 标签
- 简单的字体层级和装饰性标题样式

## 页面说明

- `index.html`：首页，展示个人介绍、项目、工作经历、教育经历和评价
- `projects.html`：项目列表页
- `articles.html`：文章列表页
- `contact.html`：联系页面，包含联系表单

## 文件结构

- `index.html`
- `projects.html`
- `articles.html`
- `contact.html`
- `style.css`：项目样式文件
- `script.js`：主题切换脚本

## 使用方式

1. 打开项目目录。
2. 直接在浏览器中打开 `index.html`。
3. 通过顶部导航访问其他页面。
4. 点击右上角按钮切换深色或浅色主题。

## 可继续改进的方向

- 为项目卡片增加真实项目链接和缩略图
- 为联系表单接入后端或第三方表单服务
- 增加当前导航高亮到所有页面
- 优化 SEO 内容为真实站点信息
- 添加 favicon 和社交分享图片
